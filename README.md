# Parish Website

A fullstack web application for a parish website built with a modern frontend and backend architecture.

Check it out: https://zupa-svetoga-petra.hr/

## 🧱 Architecture

- **Frontend:** React + Chakra UI
- **Backend:** Spring Boot (REST API)
- **API Testing:** Postman Collection included

## 📁 Project Structure
- frontend/ → React application (Chakra UI)
- backend/ → Spring Boot REST API


## ✨ Features

- Parish information and announcements
- Mass schedule and events overview
- Contact and location details
- Responsive and accessible UI
- Clean separation between frontend and backend

## 🚀 Tech Stack

**Frontend:**
- React
- Chakra UI
- JavaScript 

**Backend:**
- Java
- Spring Boot
- Spring Web (REST API)

## 🧪 API Testing

Postman collection is included: ZUPNA STRANICA.postman_collection.json


## 🎯 Purpose

This project was developed to modernize parish communication and provide a simple, user-friendly platform for sharing important information with the community.

## 📌 Status

Active development – new features and improvements may be added over time.
